# Baraka Launch Kit

This folder contains the operator pack for onboarding the first 3-5 partner stores.

Use it when:

- you approach a new store
- you prepare a pilot launch
- you need to collect store data fast
- you want a repeatable launch process instead of explaining everything manually every time

## Recommended First Pilot Setup

- 3 to 5 stores
- 8 to 20 products per store
- 1 merchant login per store
- one main point of contact per store
- one 14-day test period

## What Success Looks Like

- stores can be onboarded in under 1 day
- merchants understand redeem flow without long training
- customers can activate and redeem without confusion
- admin can monitor activations, redeem history, store analytics, and audit logs
- you collect enough feedback to improve the product before wider rollout
