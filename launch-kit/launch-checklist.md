# Launch Checklist

Use this checklist for every store before calling it live.

## Store Setup

- store created in admin
- description added
- address added
- logo added
- cover image added

## Product Setup

- at least 8 products added
- images display correctly
- prices are correct
- discounted offer logic is clear
- inventory is set correctly

## Merchant Setup

- merchant account created
- merchant login tested
- merchant knows how to redeem

## Flow Testing

- store visible in miniapp
- products visible in miniapp
- activation works
- QR payload works
- merchant preview works
- redeem works once
- repeated redeem fails as expected

## Admin Verification

- redeem history shows the event
- audit log shows merchant login / redeem
- store analytics loads

## Launch Ready

- store contact informed
- merchant credentials delivered
- pilot dates confirmed
- support contact shared
